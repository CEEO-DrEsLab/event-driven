What is This?


This prototype demonstrates one possible way to configure the code blocks.


Every block belongs to a channel, which is something like "Port 1 Touch Sensor."


Every block has a mode, which determines what other settings are available. For now, these can be seen as title-text on mouseover.


For now, there is no way to directly add or remove code blocks in this prototype. The Import Program button can be used to import a JSON string representing the code.
